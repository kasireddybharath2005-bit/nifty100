# N100 Financial Intelligence Platform

## Overview

The N100 Financial Intelligence Platform is a Streamlit-based dashboard for analyzing Nifty 100 companies. It provides financial analysis, screening, peer comparison, sector insights, capital allocation, valuation analytics, and portfolio recommendations.

---

## Technologies Used

- Python
- Streamlit
- Pandas
- Plotly
- SQLite
